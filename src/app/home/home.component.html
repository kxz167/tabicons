Introduction:

Welcome to Tabicons: the fully web sided tab organizer! This simple application is meant to help organize those unwieldly 25+ tabs-in-one-window panic searching us coders tend to do.

The mix of Microsoft Edge's integration of vertical tabs with Google's Material Icons, gives endless possibilites for organization through tab favicons.

Use:

Icon Browsing:

At the current time, there is no browse feature built into the website. Instead, please navigate to https://fonts.google.com/icons. Here, you will be able to browse all of the icons offered. Additionally, only one style (filled) is supported at the current time.

On that site at the top, you can search for icons. The "category" drop down will determine the first part of the url, and the name determines the second.

So the icon under category "image" with name "Add A Photo" can be translated as "/image/add_a_photo"
